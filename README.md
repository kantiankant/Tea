# tea

Bubble tea ordering.

## Install

```sh
chmod +x tea.sh && mv tea.sh tea && doas/sudo install -Dm755 tea /usr/local/bin (provided that it's in your path)
```

## Usage

```sh
tea.sh
```

Follow the prompts.

Orders are appended to `orders.db`.

## Dependencies

- bash
- coreutils
- grep
- awk
